HTML

HYPER TEXT MARKUP LANGUAGE - Linguagem de Marcação de Hipertexto.

Responsável pela estrutura da página, por meio de TAG's fala o que é o quê em um site.

CSS

CASCADING STYLE SHEET - Folha de Estilo em Cascatas.

Responsável pela estilização de componentes escritos em linguagem de marcação.

JAVASCRIPT

É uma linguagem de programação responsável pela interatividade do site.

Com o JavaScript os navegadores passaram a possibilitar interações do usuário.

REACT.JS

O React é uma poderosa biblioteca para construção de interfaces visuais em JavaScript.

É uma ferramenta ideal para o desenvolvimento front-end. No React, você pode adotar estruturas do HTML e do JavaScript em um só código.

BOOTSTRAP

Bootstrap é um framework front-end que fornece estruturas de CSS para a criação de sites e aplicações responsivas de forma rápida e simples.

Além disso, pode lidar com sites de desktop e páginas de dispositivos móveis da mesma forma.

NODE.JS

JS RUNTIME ENVIROMENT - Ambiente de Execução JavaScript.

É uma plataforma que permite executar JavaScript do lado do servidor.

GIT

É um sistema de controle de versão de códigos.

Permite que várias pessoas trabalhem em um programa ao mesmo tempo por meio de ramificações de modo a manter a ordem e integração de todas as partes.

GITHUB

É uma plataforma feita para hospedagem de códigos de programação.

Também é uma rede de compartilhamento e troca de conhecimento entre os profissionais desenvolvedores.
